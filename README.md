# Sales Performance Analysis using Excel

This project analyzes sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Key Tasks Performed
- Data cleaning and formatting
- Used Excel formulas (SUM, AVERAGE, IF)
- Created pivot tables for category-wise sales and quantity
- Designed bar chart and pie chart for visualization

## Insights
- Electronics category contributes the highest sales
- Clear category-wise and region-wise sales trends identified
